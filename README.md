# JavaScript Todo App ✅

A simple and responsive Todo List application built using HTML, CSS, and JavaScript.  
This app allows users to add and delete tasks, and the tasks are saved in the browser using Local Storage, so they remain available even after refreshing or closing the browser.

## 🚀 Features

- Add new todo tasks
- Delete todo tasks
- Data persistence using Local Storage
- Tasks remain after page refresh
- Simple and clean user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Browser Local Storage
